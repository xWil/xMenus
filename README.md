# xMenus
xMenus is a plugin that allows you to create menus for your server.

## API
[![javadoc](https://javadoc.io/badge2/gg.wil.xmenus/xmenus-api/javadoc.svg)](https://javadoc.io/doc/gg.wil.xmenus/xmenus-api)

Include the xMenus API in your project with Maven:
```xml
<dependency>
    <groupId>gg.wil.xmenus</groupId>
    <artifactId>xmenus-api</artifactId>
    <version>1.0.1</version>
    <scope>compile</scope>
</dependency>
```
### Features
* **Animations** - Each item can be individually animated.
* **Define once, use anywhere** - You only need to create a menu once, then you can use it anywhere, as many times as you want.
